# Cloud-Based Office Administration Management System

## Internship Project

A scalable full-stack web application for managing student internships.

## Tech Stack
* **Frontend:** React.js, Tailwind CSS, Framer Motion, Vite
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

## Installation

1. Make sure you have Node.js and MongoDB installed.
2. Run the MongoDB service.

### Backend Setup
\\\ash
cd backend
npm install
npm run dev
\\\`n
### Frontend Setup
\\\ash
cd frontend
npm install
npm run dev
\\\`n
## Features (Week 1)
* **Student Panel:** Dashboard, Submission Form, Status Tracking, Profile view
* **Mentor Panel:** Dashboard, Review Submissions (Approve/Reject), View Students
* **Database:** MongoDB schemas for Students, Mentors, and Submissions
* **API:** RESTful endpoints for CRUD operations
